# Cutopia Cozy Corner

Your cozy corner for custom cakes and delightful treats.

## About

Cutopia Cozy Corner is a bakery website showcasing our delicious custom cakes, pastries, and treats. We specialize in creating memorable desserts for all occasions.

## Tech Stack

This project is built with:

- **React** - A JavaScript library for building user interfaces
- **TypeScript** - For type-safe code
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Beautiful UI components

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd cutopia-cozy-corner
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:8080`

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run test` - Run tests
